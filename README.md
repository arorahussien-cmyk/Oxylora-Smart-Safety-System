# Oxylora Smart Safety System

## Overview
Oxylora is an intelligent multi-gas detection system designed to enhance environmental safety by detecting hazardous gases in real-time.

## Problem
Gas leaks and poor air quality can lead to explosions or suffocation, especially in closed environments. Many existing systems lack fast response and accurate classification of risks.

## Solution
Oxylora integrates multiple sensors to detect gas presence and classify hazards into:
- Explosion risk
- Suffocation risk

## Features
- Multi-gas detection using MQ-6 and MG-811 sensors  
- Real-time monitoring system  
- Hazard classification system  
- Fast response time (3–8 seconds)  
- Reduced false alarms using multi-sensor validation  

## Technologies Used
- Arduino  
- MQ-6 Gas Sensor  
- MG-811 CO2 Sensor  
- Embedded Systems Programming  

## Results
- Detection accuracy: 85–92%  
- Reliable performance in controlled testing
